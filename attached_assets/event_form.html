{% translate "Wymagania ogólne:" %}

  • {% translate "Implementacja musi być w pełni zgodna ze standardami WCAG AAA (Web Content Accessibility Guidelines)." %}
  • {% translate "Design musi być responsywny i działać poprawnie na urządzeniach mobilnych, tabletach i desktopach." %}
  • {% translate "Wszystkie elementy interfejsu muszą być zgodne z księgą znaku Uniwersytetu w Siedlcach." %}
  • {% translate "Podstawowy kolor marki to #003d7c. Użyj go jako głównego koloru akcentowego." %}
  • {% translate "Należy wykorzystać Bootstrap 5 jako bazowy framework CSS." %}

{% translate "Dane przekazywane do szablonu" %}

{% translate "Do szablonu przekazywane są następujące dane:" %}

{% translate "1. Tytuł formularza (title):" %}

  • {% translate "Zmienna 'title' - określa czy jest to formularz tworzenia ('Utwórz wydarzenie') czy edycji ('Edytuj wydarzenie')" %}

{% translate "2. Formularz wydarzenia (form):" %}

  • {% translate "Obiekt formularza Django z następującymi polami:" %}
    • form.name - {% translate "pole nazwy wydarzenia (text)" %}
    • form.description - {% translate "pole opisu wydarzenia (textarea)" %}
    • form.start_date - {% translate "pole daty i czasu rozpoczęcia (datetime)" %}
    • form.end_date - {% translate "pole daty i czasu zakończenia (datetime)" %}
    • form.location - {% translate "pole lokalizacji (text)" %}
    • form.mode - {% translate "pole trybu wydarzenia (select: online/stacjonarny/hybrydowy)" %}
    • form.organizer - {% translate "pole organizatora (text)" %}
    • form.color - {% translate "pole koloru wydarzenia (color picker)" %}
    • form.capacity - {% translate "pole pojemności wydarzenia (number)" %}
  • {% translate "Dla każdego pola formularza dostępne są następujące atrybuty:" %}
    • field.label - {% translate "etykieta pola" %}
    • field.id_for_label - {% translate "identyfikator HTML dla etykiety" %}
    • field.field.required - {% translate "czy pole jest wymagane (boolean)" %}
    • field.errors - {% translate "błędy walidacji pola (jeśli istnieją)" %}
  • form.non_field_errors - {% translate "błędy walidacji na poziomie formularza (jeśli istnieją)" %}

{% translate "Elementy do zaimplementowania" %}

{% translate "1. Nawigacja:" %}

  • {% translate "Nagłówek karty z tytułem formularza (Utwórz/Edytuj wydarzenie)" %}
  • {% translate "Formularz podzielony na logiczne sekcje z użyciem fieldsetów i legend:" %}
    • {% translate "Sekcja 'Informacje o wydarzeniu' (nazwa, opis)" %}

    • {% translate "Sekcja 'Organizacja' (organizator, kolor wydarzenia)" %}

  • {% translate "Każde pole formularza powinno zawierać:" %}

    • {% translate "Pole formularza z odpowiednimi atrybutami dostępności" %}

  • {% translate "Sekcja błędów formularza (wyświetlana, gdy wystąpią błędy na poziomie formularza)" %}
