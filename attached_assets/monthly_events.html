{% translate "Zadanie rekrutacyjne - Widok miesięczny wydarzeń" %}

{% translate "Opis zadania" %}

{% translate "Twoim zadaniem jest zaimplementowanie widoku kalendarza miesięcznego wydarzeń. Widok ten powinien prezentować wszystkie wydarzenia w danym miesiącu w formie kalendarza, umożliwiając użytkownikom łatwe przeglądanie wydarzeń w kontekście czasu." %}

{% translate "Wymagania ogólne:" %}

  • {% translate "Implementacja musi być w pełni zgodna ze standardami WCAG AAA (Web Content Accessibility Guidelines)." %}
  • {% translate "Design musi być responsywny i działać poprawnie na urządzeniach mobilnych, tabletach i desktopach." %}
  • {% translate "Wszystkie elementy interfejsu muszą być zgodne z księgą znaku Uniwersytetu w Siedlcach." %}
  • {% translate "Podstawowy kolor marki to #003d7c. Użyj go jako głównego koloru akcentowego." %}
  • {% translate "Możesz użyć dowolnego frameworka JavaScript do implementacji kalendarza (np. FullCalendar, Calendar.js itp.)." %}
  • {% translate "Bazowy framework CSS to Bootstrap 5, ale możesz dostosować go do potrzeb wybranego frameworka kalendarza." %}

{% translate "Dane przekazywane do szablonu" %}

{% translate "Do szablonu przekazywane są następujące dane:" %}

{% translate "1. Dane dotyczące miesiąca:" %}

  • month_name - {% translate "nazwa bieżącego miesiąca (string)" %}
  • year - {% translate "bieżący rok (integer)" %}
  • prev_month - {% translate "numer poprzedniego miesiąca (integer)" %}
  • prev_year - {% translate "rok dla poprzedniego miesiąca (integer)" %}
  • next_month - {% translate "numer następnego miesiąca (integer)" %}
  • next_year - {% translate "rok dla następnego miesiąca (integer)" %}
  • current_month - {% translate "numer bieżącego miesiąca (string z dopełnieniem zera)" %}
  • today_date - {% translate "dzisiejsza data w formacie YYYY-MM-DD (string)" %}

{% translate "2. Dane kalendarza:" %}

  • calendar - {% translate "dwuwymiarowa lista reprezentująca dni miesiąca pogrupowane w tygodnie" %}
  • {% translate "Wartości w kalendarzu:" %}
    • {% translate "0 - miejsce puste (dzień nie należy do bieżącego miesiąca)" %}
    • {% translate "1-31 - numer dnia miesiąca" %}

{% translate "3. Lista wydarzeń:" %}

  • events - {% translate "kolekcja obiektów wydarzeń w danym miesiącu, gdzie każde wydarzenie (event) zawiera:" %}
    • event.name - {% translate "nazwa wydarzenia (string)" %}
    • event.description - {% translate "opis wydarzenia (text)" %}
    • event.start_date - {% translate "data i czas rozpoczęcia (datetime)" %}
    • event.location - {% translate "miejsce wydarzenia (string)" %}
    • event.get_mode_display - {% translate "tryb wydarzenia (string)" %}
    • event.color - {% translate "kolor wydarzenia (string w formacie HEX)" %}
    • event.pk - {% translate "identyfikator wydarzenia (integer)" %}

{% translate "Wymagane funkcje kalendarza" %}

{% translate "Twój kalendarz wydarzeń powinien oferować następujące funkcje:" %}

{% translate "1. Główne funkcje:" %}

  • {% translate "Nagłówek strony z tytułem 'Kalendarz wydarzeń'" %}
  • {% translate "Wyświetlanie bieżącego miesiąca i roku" %}
  • {% translate "Nawigacja między miesiącami (poprzedni/następny)" %}
  • {% translate "Przycisk powrotu do bieżącego miesiąca" %}
  • {% translate "Przycisk 'Utwórz wydarzenie' przekierowujący do formularza tworzenia wydarzenia" %}

{% translate "2. Wyświetlanie wydarzeń:" %}

  • {% translate "Wyraźne oznaczenie dni z wydarzeniami" %}
  • {% translate "Wyróżnienie bieżącego dnia" %}
  • {% translate "Dla każdego wydarzenia widocznego w kalendarzu należy wyświetlić co najmniej:" %}
    • {% translate "Nazwę wydarzenia (z możliwością kliknięcia, aby zobaczyć szczegóły)" %}
    • {% translate "Godzinę rozpoczęcia" %}
    • {% translate "Wizualne oznaczenie (kolor) odpowiadające kategorii wydarzenia" %}

{% translate "3. Dodatkowe funkcje:" %}

  • {% translate "Możliwość wyświetlenia dodatkowych informacji o wydarzeniu po najechaniu myszką (tooltip) lub kliknięciu" %}
  • {% translate "Responsywne zachowanie kalendarza na różnych urządzeniach" %}
  • {% translate "Wyraźny komunikat, gdy w danym miesiącu nie ma żadnych wydarzeń" %}

  • {% translate "Przejścia między miesiącami nie mogą powodować dezorientacji użytkownika" %}
  • {% translate "Wszystkie interaktywne elementy muszą mieć odpowiednie etykiety ARIA" %}
