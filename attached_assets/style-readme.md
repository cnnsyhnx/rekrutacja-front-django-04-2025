# Style Guidelines

## Brand Colors
- Main Brand Color: #003d7c
- Secondary Colors:
  - Light Blue: #4a90e2
  - Grey: #f5f5f5
  - White: #ffffff

## Typography
- Primary Font: Roboto
- Secondary Font: Open Sans
- Headings: Roboto Bold
- Body Text: Open Sans Regular

## Components
- Buttons should have rounded corners (border-radius: 4px)
- Forms should have clear labels and validation messages
- Icons should be consistent throughout the application
- Cards should have subtle shadows for depth

## Accessibility Guidelines
- All UI components must be accessible (WCAG AAA compliant)
- Ensure proper contrast ratios for text
- Include proper ARIA attributes
- Ensure keyboard navigability
- All text should be translatable

## Layout
- Use Bootstrap 5 grid system
- Responsive design for all screen sizes
- Consistent spacing and margins

## Forms
- Clear labels for all form fields
- Inline validation where possible
- Consistent button styling and placement
- Required fields clearly marked 

## JavaScript Frameworks
- Możliwe jest wykorzystanie dowolnych frameworków JavaScript (React, Vue, Angular, itp.)
- Kalendarze mogą być implementowane przy użyciu dedykowanych bibliotek (FullCalendar, Calendar.js, itp.)
- Konieczne jest zachowanie spójności stylów i dostępności niezależnie od użytych bibliotek
- Wszystkie rozwiązania muszą być w pełni responsywne
- W przypadku użycia frameworków konieczne jest dołączenie instrukcji integracji z obecnym projektem
